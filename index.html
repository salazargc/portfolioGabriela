    WebGIS Weather Application

    This weather app was developed using open-source web mapping libraries like Leaflet.js, along with HTML, CSS, and Vanilla JavaScript. It integrates the Gemini API and OpenWeatherMap API to display real-time weather information and AI-generated weather suggestions. The app retrieves data such as the current temperature, wind speed, humidity, and weather forecasts for the upcoming days in a selected city. Additionally, it provides personalized suggestions, including recommended activities, precautions, and advice based on weather conditions. When a city name is entered, the app zooms in on its geographical location and opens a popup displaying relevant weather insights. This app is designed to be useful for travel planning, environmental monitoring, and survey planning, among other applications.

    WebGIS Trails Management Application

    This application was built using the ArcGIS Maps SDK for JavaScript as the mapping library, along with HTML, CSS, and vanilla JavaScript. It provides users with information about trail locations, including details on accessibility. Users can query data for specific parks, view trail usage statistics displayed in a chart, bookmark areas of interest, and print the information.
